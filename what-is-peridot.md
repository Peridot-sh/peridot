# What is Peridot?

Peridot is a self-hosted AI chat, search, and application platform that runs inside your cloud (AWS, GCP, Azure), giving enterprises full control over data, models, and usage.

---

## The Problem

Most AI tools optimize for speed.

But enterprises quickly run into issues:

- Data leaving their environment [data leakage in AI](../questions/how-to-prevent-data-leakage-in-ai.md)
- No control over which LLMs are used  
- Shadow AI across teams  
- Lack of visibility into usage  
- Rising and unpredictable costs  

These issues create **real risk at scale**.

Learn more about how this impacts security → [Peridot Security](security/)

---

## What Peridot Does

Peridot acts as a control layer for AI inside your organization.

It allows teams to:

- Build AI applications (chat, copilots, workflows)  
- Run everything inside their own cloud  
- Choose and switch between LLMs  
- Control how data flows between systems  
- Govern who can build and use applications  
- Monitor usage and costs  

See how this works in practice → [Use Cases](use-cases/)

## When to Use Peridot

Peridot is best suited for:

- Teams building internal AI applications  
- Companies working with sensitive data  
- Organizations scaling AI across multiple teams  
- Enterprises needing governance and compliance
